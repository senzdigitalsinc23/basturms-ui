# **App Name**: CampusConnect

## Core Features:

- Multi-Role Authentication: Secure authentication supporting distinct roles: Admin, Teacher, Student, and Parent.
- Role-Based Access Control: RBAC middleware to manage and enforce user access permissions based on assigned roles.
- User Management (CRUD): Admin interface for creating, reading, updating, and deleting user accounts (Teachers, Students, Parents).
- Basic Audit Logs: Maintain a log of user actions within the system for security and debugging purposes.
- User Onboarding Tips: Provide in-app tips, instructions, and contextual help using a generative AI tool to create hints on the fly to reduce the burden on system admins.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to convey trust and professionalism.
- Background color: Very light lavender (#F0F8FF) to ensure comfortable readability and a calm feel.
- Accent color: Vivid purple (#9400D3) to highlight interactive elements.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines, 'Inter' (sans-serif) for body text.
- Use clean, minimalist icons to represent actions and entities, maintaining a consistent style throughout the interface.
- Maintain a consistent, card-based layout with adequate spacing to avoid a cluttered feel, prioritizing important data.
- Subtle transitions and feedback animations to acknowledge user interactions, enhancing the overall experience.
